# homework_final
数字素养结课作业
商品销售数据分析项目
一、项目背景
零售业态的竞争焦点已由“渠道”转向“精细化运营”。本课程以运动服饰电商2022年全量订单为研究对象，通过Excel函数构建分析框架，回答两个管理命题：
商品整体销量在月度维度上的波动特征；
各小类商品是否存在显著的季节或促销节奏差异。
结论可直接服务于补货计划、营销排期及资金预算。
二、数据概况
信息表：60条记录，字段包括商品大类、小类、名称、编号、销售价，建立层级关系。
销售数据表：2000条记录，字段包括订单日期（Excel序列值）、订单编号、商品编号、订单数量，覆盖2022-01-01至2022-12-31，无缺失、无负值。
三、分析方案
序列值转换
利用Excel内置日期系统，将序列值转换为常规日期，并提取月份字段。
汇总机制
以SUMIFS函数为核心，先按月份聚合整体销量，再按“小类+月份”二维条件聚合细分销量。
可视化
折线图：呈现连续趋势；
柱状图：比较品类差异；
条件格式热力图：快速识别淡旺季交叉点。
全程在Excel内完成，无需外部脚本。
四、主要结论
月度总量
春节所在2月形成全年低谷，环比下滑约38%；3月复工复学后迅速回补；7月因暑期运动需求达到峰值；9月开学季出现次高峰；11月受“双11”促销拉动，12月因保暖需求维持高位。
小类特征
背心：旺季始于5月，7月登顶，季节弹性最强；
长裤：10月后显著放量，与气温下降高度同步；
配饰（袜、发带、手环）：全年波动最小，需求相对刚性；
外套：11月销量占自身全年28%，对促销节奏高度敏感。
管理启示
背心补货窗口应前置至4月；长裤需在9月完成增仓；配饰安全库存可下调10%以释放资金；外套应平衡折扣深度与毛利贡献，避免高销量低利润。

风险控制
